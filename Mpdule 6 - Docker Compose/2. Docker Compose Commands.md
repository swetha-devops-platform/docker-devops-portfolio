| Category          | Command                                            | Description                                      |
| ----------------- | -------------------------------------------------- | ------------------------------------------------ |
| **Start & Stop**  | `docker compose up`                                | Start all services in foreground                 |
|                   | `docker compose up -d`                             | Start all services in background                 |
|                   | `docker compose down`                              | Stop and remove containers                       |
|                   | `docker compose down -v`                           | Stop and remove containers and volumes           |
|                   | `docker compose down --rmi all`                    | Stop and remove containers and images            |
|                   | `docker compose start`                             | Start stopped services                           |
|                   | `docker compose stop`                              | Stop running services but keep containers        |
|                   | `docker compose restart`                           | Restart all services                             |
| **Build**         | `docker compose build`                             | Build all images                                 |
|                   | `docker compose build backend`                     | Build a specific service                         |
|                   | `docker compose up --build`                        | Build images and start services                  |
| **Status & Logs** | `docker compose ps`                                | List all services and their status               |
|                   | `docker compose logs`                              | View logs of all services                        |
|                   | `docker compose logs -f`                           | Follow logs in real time                         |
|                   | `docker compose logs backend`                      | View logs of a specific service                  |
|                   | `docker compose top`                               | Show running processes inside services           |
| **Scale**         | `docker compose up -d --scale backend=3`           | Run 3 instances of backend service               |
| **Exec & Run**    | `docker compose exec backend bash`                 | Open shell inside running backend container      |
|                   | `docker compose exec database psql -U admin myapp` | Open PostgreSQL shell inside database container  |
|                   | `docker compose run backend npm test`              | Run one-time command in backend service          |
| **Pull & Push**   | `docker compose pull`                              | Pull latest images from registry                 |
|                   | `docker compose push`                              | Push built images to registry                    |
| **Config**        | `docker compose config`                            | Validate and display final compose configuration |
