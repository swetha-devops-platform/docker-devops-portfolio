# Complete Docker Commands Cheat Sheet

| Category                 | Command                              | Description                              |
| ------------------------ | ------------------------------------ | ---------------------------------------- |
| **Image Management**     | `docker pull ubuntu`                 | Download image from Docker Hub           |
|                          | `docker images`                      | List all local images                    |
|                          | `docker build -t myapp .`            | Build image from Dockerfile              |
|                          | `docker tag myapp myuser/myapp:v1.0` | Tag an image                             |
|                          | `docker push myuser/myapp:v1.0`      | Push image to registry                   |
|                          | `docker pull myuser/myapp:v1.0`      | Pull image from registry                 |
|                          | `docker rmi ubuntu`                  | Remove Docker image                      |
|                          | `docker image prune`                 | Remove unused images                     |
| **Container Lifecycle**  | `docker create myapp`                | Create container without starting        |
|                          | `docker run ubuntu`                  | Create and run container                 |
|                          | `docker run -d -p 80:80 nginx`       | Run container in background and map port |
|                          | `docker start mycontainer`           | Start stopped container                  |
|                          | `docker stop mycontainer`            | Gracefully stop container                |
|                          | `docker restart mycontainer`         | Restart container                        |
|                          | `docker kill mycontainer`            | Force stop container                     |
|                          | `docker pause mycontainer`           | Pause container processes                |
|                          | `docker unpause mycontainer`         | Resume paused container                  |
|                          | `docker rm mycontainer`              | Remove container                         |
|                          | `docker rename oldname newname`      | Rename container                         |
| **Container Monitoring** | `docker ps`                          | List running containers                  |
|                          | `docker ps -a`                       | List all containers                      |
|                          | `docker logs mycontainer`            | View container logs                      |
|                          | `docker top mycontainer`             | Show running processes                   |
|                          | `docker stats mycontainer`           | Show CPU and memory usage                |
|                          | `docker inspect mycontainer`         | Detailed container information           |
| **Container Access**     | `docker exec -it <id> bash`          | Enter running container                  |
|                          | `docker attach mycontainer`          | Attach terminal to container             |
| **Networking**           | `docker network ls`                  | List Docker networks                     |
|                          | `docker network create mynet`        | Create Docker network                    |
|                          | `docker network inspect mynet`       | Inspect network details                  |
|                          | `docker network rm mynet`            | Remove Docker network                    |
| **Volume Management**    | `docker volume ls`                   | List Docker volumes                      |
|                          | `docker volume create myvolume`      | Create Docker volume                     |
|                          | `docker volume inspect myvolume`     | Inspect Docker volume                    |
|                          | `docker volume rm myvolume`          | Remove Docker volume                     |
| **Docker Registry**      | `docker login`                       | Login to Docker Hub                      |
|                          | `docker logout`                      | Logout from Docker Hub                   |
|                          | `docker search nginx`                | Search images on Docker Hub              |
| **Docker System**        | `docker info`                        | Show Docker system information           |
|                          | `docker version`                     | Show Docker version                      |
|                          | `docker system df`                   | Show Docker disk usage                   |
|                          | `docker system prune`                | Remove unused Docker resources           |
|                          | `docker system prune -a`             | Remove all unused images and containers  |
| **Docker Compose**       | `docker compose up -d`               | Start services in background             |
|                          | `docker compose down`                | Stop and remove services                 |
|                          | `docker compose ps`                  | List compose services                    |
|                          | `docker compose logs`                | View compose logs                        |
|                          | `docker compose restart`             | Restart compose services                 |
