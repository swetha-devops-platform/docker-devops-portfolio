Step 4 — Start & Enable Docker

  - **sudo systemctl start docker**
    
  - **sudo systemctl enable docker**

---

<img width="1033" height="89" alt="image" src="https://github.com/user-attachments/assets/9a86a497-ed29-44b0-9cf9-1181ce515293" />


---

Step 5 — Add User to Docker Group - **sudo usermod -aG docker ubuntu**

  - By default, Docker commands require sudo every time. To run Docker commands without sudo, you need to add your user to the Docker group.

  - Ubuntu is the username 

  - Docker group is automatically created when Docker is installed.

  - need to logout and login back for the changes to be reflected.
