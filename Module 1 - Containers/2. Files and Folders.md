#  Files and Folders in Container Base Images

| Directory | Description                                                            |
| --------- | ---------------------------------------------------------------------- |
| `/bin`    | Contains binary executable files such as `ls`, `cp`, and `ps` commands |
| `/sbin`   | Contains system binary executable files such as `init` and `shutdown`  |
| `/etc`    | Stores configuration files for various system services                 |
| `/lib`    | Contains shared library files used by binaries                         |
| `/usr`    | Contains user-related applications, libraries, and documentation       |
| `/var`    | Stores variable data such as logs, spool files, and temporary files    |
| `/root`   | Home directory of the root user                                        |

#  Resources Containers Use from Host Operating System

| Resource                 | Description                                                                     |
| ------------------------ | ------------------------------------------------------------------------------- |
| Host File System         | Containers can access host files using bind mounts for reading and writing data |
| Networking Stack         | Containers use the host networking stack for network communication              |
| System Calls             | Containers use the host kernel to access CPU, memory, and I/O resources         |
| Namespaces               | Provide isolation for processes, networking, file systems, and process IDs      |
| Control Groups (cgroups) | Limit and manage container resource usage such as CPU, memory, and I/O          |
