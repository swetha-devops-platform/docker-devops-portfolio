# Docker Networking

**Demo Overview**

This demo demonstrates Docker networking concepts by creating three containers — Entry, Bank, and Account — setting up a custom bridge network for the Account container, and verifying connectivity between containers using ping tests.

---

**Concepts Covered**

  - Default Bridge Network

  - Custom Bridge Network

  - Container-to-Container Communication

  - Network Inspection & Ping Testing
