| Category                 | Command                                                                                                             | Description                                          |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------- |
| **List & Inspect**       | `docker network ls`                                                                                                 | List all Docker networks                             |
|                          | `docker network inspect bridge`                                                                                     | Inspect default bridge network                       |
|                          | `docker network inspect mynetwork`                                                                                  | View containers connected to a network               |
| **Create Network**       | `docker network create mynetwork`                                                                                   | Create a default bridge network                      |
|                          | `docker network create --driver host mynet`                                                                         | Create network using host driver                     |
|                          | `docker network create --driver bridge --subnet 10.10.0.0/16 --gateway 10.10.0.1 --ip-range 10.10.1.0/24 mynetwork` | Create custom bridge network with subnet and gateway |
| **Connect & Disconnect** | `docker network connect mynetwork mycontainer`                                                                      | Connect container to network                         |
|                          | `docker network disconnect mynetwork mycontainer`                                                                   | Disconnect container from network                    |
| **Remove Network**       | `docker network rm mynetwork`                                                                                       | Remove a specific network                            |
|                          | `docker network prune`                                                                                              | Remove all unused networks                           |
| **Debugging**            | `docker exec mycontainer ip addr`                                                                                   | View container IP addresses                          |
|                          | `docker exec mycontainer ip route`                                                                                  | View routing table inside container                  |
|                          | `docker exec mycontainer cat /etc/resolv.conf`                                                                      | View DNS configuration                               |
|                          | `docker exec mycontainer ping other-container`                                                                      | Test connectivity between containers                 |
|                          | `docker exec mycontainer netstat -tulpn`                                                                            | View open ports and listening services               |
